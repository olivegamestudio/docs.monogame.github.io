# Requirements

## Android Development

**Android** development can be accomplished on either a Windows PC or Mac. However, ARM64 variants of Windows do not run the Android simulator well.

**Android Device Manager** - Used to set up simulators. This is accessible through Visual Studio.

**Android SDK Manager** - Used to install the SDK platforms.

## iOS Development

For **iOS** development, you will need a Mac, whether you choose to develop entirely on it or use the Pair to Mac option in Visual Studio.

Since **Visual Studio for Mac** has been deprecated, the currently recommended development software for Mac is [JetBrains Rider](https://www.jetbrains.com/rider/). Visual Studio Code is another viable option.

**Xcode** - Required for iOS development and deployment.

To deploy to a physical device, you will need an Apple Developer account: [https://developer.apple.com/programs/enroll/](https://developer.apple.com/programs/enroll/)
